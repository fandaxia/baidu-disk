# baidu-disk

>  div+css布局高仿还原百度网盘页面（单页应用，虚拟数据）  
jQuery实现文件区操作的主要功能：  
        文件图标根据数据动态渲染  
        视图模式切换  
        文件的移入移出单选多选效果  
        文件区顶部信息实时更新  
        右键菜单（根据不同情况显示不同菜单）  
        菜单功能：打开文件夹，复制粘贴功能，重命名，删除，新建文件夹等  
        顶部路由

#### 在线地址：http://qiandu.duapp.com

## 技术栈
- jQuery html css javascript
